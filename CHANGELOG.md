## [1.0.0] — 2024-01-01

### Добавлено

- **JWE Compact Serialization** (RFC 7516) — полная реализация без сторонних зависимостей
- **RSA-OAEP-256** шифрование ключа (RFC 8017 / PKCS #1 v2.2)
  - OAEP паддинг с MGF1(SHA-256)
  - Арифметика больших чисел (`bigint.hpp`) — pure C++23
- **AES-256-GCM** шифрование данных (FIPS 197 + NIST SP 800-38D)
  - AES-256 block cipher — полная реализация с нуля
  - GHASH аутентификатор (GF(2¹²⁸))
  - 96-битный IV, 128-битный тег аутентификации
- **SHA-256** и **HMAC-SHA-256** (FIPS 180-4, RFC 2104)
  - Инкрементальный интерфейс (`update()` / `finalize()`)
  - MGF1(SHA-256) для OAEP
- **BASE64URL** кодирование/декодирование (RFC 4648 §5, без паддинга)
- **JSON-парсер** — минимальный, достаточный для JWKS (RFC 7517)
- **JWKS fetcher** — загрузка публичных ключей по HTTP URL
- **JweBuilder** — fluent API для построения токенов
- **CLI-утилита** `jwe_encrypt` — шифрование из командной строки
- **Демонстрационный пример** `basic_usage` — пошаговый вывод всех компонентов
- **46 модульных тестов** на официальных NIST/RFC тест-векторах
- **Dockerfile** — многоэтапная сборка, тесты встроены в образ
- **GitHub Actions CI** — Linux (GCC 13, Clang 17), macOS, Docker, ASan

### Технические требования

- C++23 (GCC 13+, Clang 17+, MSVC 2022+)
- CMake 3.20+
- Только стандартная библиотека C++ + POSIX (для `/dev/urandom` и сокетов)
